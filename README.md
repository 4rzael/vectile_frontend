# vectile_frontend
A really basic leaflet app to render vector tiles served by TileStache, for example

## CLI

* -o : Output port. Default `3000`

* -h : Tile host. Default `localhost`

* -p : Tile port. Default `8080`

* -l : Tile layer. Default `GIS`

* --help : help
